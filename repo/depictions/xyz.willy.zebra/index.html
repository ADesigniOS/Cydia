CyStore

  • Supports iOS 11.0 to 12.1.2

Description

  • CyStore is a very basic package manager for unlocked iOS devices.

     

    It supports most modern iOS versions and provides most of the features that Cydia does, while noticeably faster.

     

    This product is still in beta and I thank you for your help in testing it. If there are any problems you encounter, features you want to request or anything else, please send me and send via twitter and I will contact you shortly.
